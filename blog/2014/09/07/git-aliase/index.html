让Git命令更简单（Git Alias）

简单使用 git 命令的方式莫过于添加别名了，alias时linux下一个常用命令详见：http://en.wikipedia.org/wiki/Alias_(command)

如何添加别名

假设我们想用 git ci代替git commit, 我们可以添加如下命令到 ~/.gitconfig 文件中

［alias］
    ci = commit

如果不习惯手动编辑 config 文件，也可以使用命令 git config alias.ci commit来代替，如果想在自己机器的任何地方都使用改别名，则可添加 --global 标记

git config --global alias.ci commit

常用别名

[alias]
  st = status
  ci = commit
  br = branch
  co = checkout
  df = diff
  ad=add
  cp = cherry-pick
  lg = log -p

其他别名

1.初始化git仓库

this = !git init && git add . && git commit -m \"initial commit\"

2.暂存

sl = stash list

sa = stash apply

ss = stash save

2.删除已经删除的文件

rd = git rm $(git ls-files --deleted)

3.清空未暂存的文件

cd = git clean -df

4.撤销本地所有修改

cl = git checkout .

5.列出所有别名

alias = config --get-regexp 'alias.*'
