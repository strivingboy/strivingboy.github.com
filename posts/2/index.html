Git 使用基础(二)

git checkout

git checkout 命令提供三种功能：1、查看文件历史修改 2、切换到某次提交（或某次提交的特定文件） 3、切换分支

用法：

git checkout master

从当前分支切换到master分支

git checkout <commit> <file>

将本地文件file 更新到 commit 次提交

对于git的学习推荐一下资料：

  • GitPro:http://git-scm.com/book/zh 很不错的入门书籍

http://www.sourcetreeapp.com/ 各个平台下都有，更新也非常快
